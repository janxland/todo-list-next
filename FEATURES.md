## 🏗️ 架构特点

### 松耦合高内聚
- **组合式 Hooks**: `useTodos`, `useStats`, `useImportExport`, `useTheme`, `useCategoryManagement`
- **组件化设计**: 每个功能模块独立，职责单一
- **类型安全**: 完整的 TypeScript 类型定义
- **CSS变量主题**: 统一的主题管理系统

### 文件结构
```
src/
├── app/                    # 页面路由
│   ├── page.tsx           # 首页(含拖拽功能)
│   ├── stats/             # 统计页面
│   ├── import-export/     # 数据管理页面
│   └── categories/        # 分类管理页面
├── components/            # 组件库
│   ├── home/             # 首页组件
│   ├── stats/            # 统计组件
│   ├── import-export/    # 导入导出组件
│   ├── categories/       # 分类管理组件
│   └── ui/               # 通用UI组件
├── hooks/                # 组合式Hooks
├── lib/                  # 工具库
└── types/                # 类型定义
```
